## Technical Architecture & Design Decisions
* **Zero Backend Attack Surface:** Hosted statically on a decentralized CDN (GitHub Pages) to mitigate traditional server-side vulnerabilities (SQLi, Directory Traversal, RCE). 
* **Liquid Glass UI:** Utilizes CSS `backdrop-filter: blur()` properties to achieve a highly responsive, translucent Glassmorphism interface over high-definition media, prioritizing readability and premium branding.
* **AI-Accelerated Workflows:** Developed using an AI co-pilot, showcasing a proactive embrace of modern LLM integration while retaining final architectural sign-off and engineering oversight.
